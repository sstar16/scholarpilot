# 万方数据检索规则

## API 事实

- 中文学术资源数据库，覆盖中文期刊、学位论文、会议论文
- 中文关键词为主，空格分隔默认 AND
- **不支持**英文术语（会返回无关或空结果）

## 适合主题

- 中文学术文献（期刊、学位论文、会议）
- 中文研究项目必选
- 中文 CNKI 等的补充

## 关键词生成规则

**必须**：2-4 个精准中文学术术语
**推荐**：
- 用学术论文的正式中文术语
- 避免口语化和英文缩写
- 化学物质用中文名

**禁止**：
- 英文
- 超过 4 个词
- 停用词（"的"、"研究"、"本文"、"方法"、"分析"）

## 示例

| 研究主题 | 推荐 query |
|---|---|
| 深度学习推理加速 | `深度学习 推理 加速 模型压缩` |
| 新能源汽车电池 | `动力电池 磷酸铁锂 正极` |
