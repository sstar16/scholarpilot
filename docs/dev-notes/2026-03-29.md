## Session 1 — MVP 跑通端到端流程

### 目标

跑通 MVP 端到端检索流程：创建项目 → Round 1 检索 → AI摘要 → 反馈提交 → Round 2 自动启动。

### 修复的 Bug

#### 1. PubMed 摘要为空（已知，暂绕过）

**根因**：`eutils.ncbi.nlm.nih.gov` 在国内 Docker/WSL2 环境被 GFW 封锁，TLS 握手超时。
**修复**：`DISABLED_SOURCES=pubmed`，改用 EuropePMC（内容相同，有完整摘要，可从国内访问）。
**额外改进**：所有 fetcher 异常改为打印 `traceback.format_exc()`，便于后续排查。

#### 2. 所有来源返回 0 篇文档

**根因**：LLM 把描述翻译成 6 个多词短语后直接 join，形成 22 词长查询字符串。OpenAlex/EuropePMC 对极精确查询返回 0 结果。
**修复**：`query_builder._get_english_query()` 将 LLM 返回关键词展开为单词列表取前 4 个。
**教训**：学术 API 的全文搜索越短越好，3-4 词查询的召回率远高于 8+ 词。

#### 3. SemanticScholar 持续 429

**根因**：SS 公开 API 无 API Key 时限速极严。
**修复**：4 次指数退避重试；将 `semantic_scholar` 加入 `DISABLED_SOURCES` 默认禁用。

#### 4. arXiv 超时

**根因**：`export.arxiv.org` 在 Docker 容器内被封锁。
**修复**：`DISABLED_SOURCES` 加入 `arxiv,biorxiv,medrxiv`。

#### 5. nginx 502（容器重建后）

**根因**：nginx 启动时解析并缓存后端 IP，容器重建后 IP 变化但 nginx 不感知。
**修复**：每次 `up -d` 后必须执行 `docker-compose restart nginx`。

#### 6. `positive_embedding` 列不存在

**根因**：ORM 模型定义了 `Vector(384)` 列，但 alembic 迁移文件不存在，且目录未挂载。
**修复**：应急用 `ALTER TABLE`，之后创建正式迁移 + 挂载 alembic 目录到 volume。
**教训**：新增 ORM 列后必须同步创建迁移文件，迁移目录必须挂载。

#### 7. 反馈提交后前端不自动推进到下一轮

**根因**：`submitFeedback()` 没有为新 round 启动轮询；刷新页面后也不恢复。
**修复**：`stores/search.ts` 中 `fetchRounds()` 检测到 running/pending round 时自动调用 `startPolling()`。

#### 8. `proxy=None` 阻断系统代理

**根因**：httpx `AsyncClient` 显式写 `proxy=None`，禁用了系统代理。
**修复**：删除所有 fetcher 中的 `proxy=None` 参数。

### 架构决策

- **DISABLED_SOURCES 环境变量**：运行时过滤数据源，无需改代码。
- **EuropePMC 替代 PubMed**：内容相同，对国内网络友好。
- **查询长度 4 词上限**：3 词 → 6391 条，8 词 → 0 条，4 词是最佳平衡点。

### 临时公网访问

使用 Cloudflare Quick Tunnel：
```bash
cloudflared tunnel --url http://localhost
```
注意：每次启动 URL 不同，仅供临时演示。

### 用户反馈处理

| 反馈 | 根因 | 修复 |

| 领域只能单选 | `el-select` 无 `multiple` | 改为多选，支持同时选多个领域 |

### 修复的 Bug

#### el-table 内 el-select 数字绑定失效

**根因**：Vue 3 + Element Plus 中，在 `el-table` scoped slot 里用 `:value="10"`（数字）绑定 `el-select`，v-model 比较失效，所有行都显示第一个选项。
**修复**：改为字符串 key（`value="10"`），提交时 `parseInt` 转回数字。
**教训**：`el-table` scoped slot 内的 `el-select` 必须用字符串 value，不能用数字或 null。

#### 摘要为空时无限 loading

**根因**：`llm_summarizer.py` 对无摘要文献（abstract < 50 字）返回 `None`，前端不知道轮次已完成，一直显示 skeleton。
**修复**：`DocumentCard` 接受 `roundStatus` prop，轮次为 `awaiting_feedback` 时显示"原文无摘要信息"。

#### 文献数 ≤ 3 时评分卡住

**根因**：最低评分数硬编码为 3，单轮只检索到 1 篇时用户无法提交。
**修复**：`min_required = total_docs if total_docs <= 3 else 3`，动态计算。

### 新增功能

- **Lens.org 全球专利 Fetcher**：覆盖 169,849,476 条专利记录，标题+摘要+权利要求三路搜索。需配置 `LENS_API_TOKEN`（免费14天试用，[lens.org/lens/user/subscriptions](https://www.lens.org/lens/user/subscriptions)）。
- **每轮独立配置表格**：创建项目时可为每轮单独设置年份范围/语言优先/返回数量，轮数增减时自动同步行数。
- **Cloudflare 临时隧道对外共享**：`cloudflared tunnel --url http://localhost` 一行命令获得公网 URL。

### 数据持久化警告

`docker-compose down -v` 会删除所有数据库数据（pgdata volume）。正确停服方式：
```bash
docker-compose down      # 安全，数据保留
docker-compose stop      # 更安全
```
